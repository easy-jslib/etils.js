# etils.js

一个 javascript 常用工具方法库

## 使用者指南

通过 npm 下载安装代码

```bash
npm install etils.js --save
```

如果使用浏览器环境

```html
<script src="node_modules/etils.js/dist/index.aio.js"></script>
<script>
  clone({ a: 1 });
</script>
```

## 贡献者指南

首次运行需要安装依赖，建议使用[pnpm](https://www.pnpm.cn/)安装

```bash
pnpm i
```

一键打包生成生产代码

```bash
pnpm build
```

运行单元测试

```bash
pnpm test
```
